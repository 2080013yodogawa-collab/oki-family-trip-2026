## 2. おすすめスポット: 西郷港岸壁釣り体験(ガイド付き)

### 基本情報

| 項目 | 内容 |
|------|------|
| 場所 | 西郷港 ポートプラザ1F前の岸壁 |
| 期間 | 4月1日〜11月30日 |
| 時間枠 | 1枠目 08:40-10:10 / 2枠目 12:30-14:00 / 3枠目 15:30-17:00 |
| 所要時間 | 90分 |
| 料金 | ガイド付き 3,000円 / 竿レンタルのみ 2,000円(餌込み) |
| 延長 | +1,000円/30分 |
| 予約 | **要予約** 隠岐旅行社 TEL: 08512-2-7100 |

### 子供に嬉しいポイント
- スタッフが**餌付け・魚外し**を全てやってくれる
- 竿・仕掛け・餌は全てレンタル(手ぶらOK)
- 岸壁は**平坦・柵あり**で3歳児でも安全
- 完全初心者向けの設計
- 釣った魚はその場で見られる(リリース or 持ち帰り)

### 推奨プラン
- **1枠目(08:40-10:10)**がおすすめ
  - 朝の方が魚の活性が高い
  - 午後は3歳児のお昼寝に充てられる
  - 日差しがまだ強くない

### 持ち物(親が用意するもの)
- [ ] 長靴(水しぶき対策)
- [ ] 着替え一式(汚れる前提)
- [ ] タオル(手拭き用 x 2)
- [ ] 帽子 + 日焼け止め
- [ ] 飲み物・おやつ
- [ ] ウェットティッシュ
- [ ] カメラ(釣れた瞬間を撮影!)
- [ ] ジップロック(魚を持ち帰る場合)

## 4. 子供用装備ガイド

### 7歳児
- ガイド付き体験で十分楽しめる
- 自分で竿を持ち、リールを巻ける年齢
- 「自分で釣った!」という達成感が重要
- 90分は集中力的にちょうど良い長さ

### 3歳児
- 竿を持つのは難しい → **見学 & お手伝い係**
- バケツの魚を観察する係にすると喜ぶ
- 飽きたら岸壁散歩(もう1人の大人が付き添い)
- 石を海に投げるだけでも楽しい時期
- **ライフジャケット推奨**(万が一に備えて)

## 5. 釣った魚の活用

### キャンプ場で調理する場合
- アジ → 塩焼き or なめろう(大人向け)
- カサゴ → 味噌汁(絶品!)
- メバル → 煮付け
- キス → 天ぷら

### 持ち物(調理する場合)
- [ ] クーラーボックス + 氷
- [ ] 包丁 + まな板
- [ ] 鱗取り
- [ ] キッチンペーパー
- [ ] ジップロック(大)

## 6. 天候別プラン

| 天気 | 釣り判断 | 代替案 |
|------|----------|--------|
| 晴れ | GO! | - |
| 曇り | GO!(むしろ好条件) | - |
| 小雨 | ガイドに要相談 | 隠岐自然館(港2F)で魚の勉強 |
| 強風・大雨 | 中止 | 五箇創生館 or 隠岐郷土館 |

> 雨天中止の場合、Day 5(5/8)に振替可能(行程に予備日あり)
