## 6. フェリー乗船メモ

| 項目 | 内容 |
|------|------|
| 車両長制限 | 長さで料金区分(要車検証) |
| 高さ制限 | **3.8m以下** |
| 幅2.5m超 | 25cmごとに15%割増 |
| 最低地上高 | **15cm以上** |
| チェックイン | **出港1時間前** |
| 乗船時 | 車内に人は乗れない(徒歩で客室へ) |
| 船内設備 | 自販機・売店・座敷席あり |
